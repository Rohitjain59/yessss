# Elyse Residence Clone

This is a pixel-perfect clone of the Elyse Residence design, built with React and Vite.

## Features
- **Hero Section**: Immersive full-screen visual with large typography.
- **About Section**: Grid layout with overlapping text and images.
- **Projects Section**: Carousel-style project showcase.
- **Responsive Design**: Adapts to different screen sizes.
- **Animations**: Smooth transitions and hover effects.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Technologies
- React
- Vite
- Vanilla CSS (Modern CSS3 features)
- Google Fonts (Cormorant Garamond & Montserrat)
